# Duggan Community League

This GitHub organization hosts Duggan Community League technology projects, including MyDCL, LeagueOS, emergency/status tooling, and reusable community-league software.

## Working Principles

- Protect member, volunteer, donor, and operational data.
- Use pull requests for changes to protected branches.
- Keep production writes, payment actions, member exports, push/email/SMS/social sends, and third-party account changes behind explicit approval.
- Prefer small, reviewable changes with clear rollback notes.

## Core Repositories

- `leagueos-stack`: core LeagueOS stack and shared local services.
- `dcl-os`: Duggan Community League operating system work.
- `leagueos-white-label`: reusable white-label community league tooling.
- `leagueos-emergency`: public emergency and incident tooling.
- `leagueos-status-origin`: public status origin tooling.
